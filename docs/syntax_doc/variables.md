## Variables

Currently, SLAP only supports one variable type: `let` (I might add immutable `const` in the future), which is mutable. Alternatively, you can use `$`.

```
let name = "bichanna";
variable = "Nobuharu Shimazu";

$age = 15;
age = age + 1;
```
